NEXUS is a premium, high-fidelity e-commerce landing page concept for next-generation computing. Built with a "Glassmorphism" aesthetic and integrated Three.js visualizations, it offers a cinematic shopping experience where hardware feels as advanced as the software running it.

 Features
Cinematic UI: A deep-space color palette utilizing Tailwind CSS, backdrop blurs (glassmorphism), and neon accents.

Interactive 3D Assembly: Features a Three.js canvas where laptop components "assemble" and "disassemble" based on user interaction and hover states.

Dynamic Product Engine:

Category filtering (Gaming, Ultrabooks, Workstations).

Real-time price sorting.

Instant search functionality for "Quantum Models."

Seamless Cart System: A slide-out drawer with quantity controls, real-time total calculation, and "Quantum Realm" checkout simulation.

Responsive Design: Fully optimized for mobile, tablet, and ultra-wide desktop displays.

Tech Stack
HTML5 / CSS3: Custom animations and scanline overlays.

Tailwind CSS: For rapid, utility-first futuristic styling.

Three.js: Powers the 3D laptop model and real-time assembly animations.

Vanilla JavaScript (ES6+): Manages state for the shopping cart, filtering, and 3D rendering loops.
